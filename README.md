# Skypro Expenses

Приложение для контроля расходов на React.

## Функционал

- Добавление нового расхода
- Фильтрация по категории
- Сортировка по дате
- Удаление расхода
- Адаптивный дизайн

## Локальная разработка

```bash
npm install
npm start
```

## Docker развертывание

### Локальная сборка и запуск

```bash
docker-compose up --build
```

Приложение будет доступно на http://localhost:3009

### Развертывание на сервер

1. Убедитесь, что на сервере установлены Docker и Docker Compose
2. Создайте директорию на сервере: `mkdir -p /home/denis/skypro`
3. Скопируйте файлы на сервер
4. Запустите на сервере:

```bash
cd /home/denis/skypro
docker-compose up -d --build
```

Или используйте скрипт развертывания:

```bash
./deploy.sh
```

Приложение будет доступно на http://109.61.108.37:3009
